评价指标

模型训练完成后，需要使用评价指标判断当前单特征模型对 MEDV 的解释能力和预测误差。

1. MSE：均方误差

MSE = (1/n) × Σ(ŷᵢ - yᵢ)²

MSE 越小，说明预测误差整体越小。由于误差被平方，大误差会被明显放大。

2. RMSE：均方根误差

RMSE = √MSE

RMSE 把 MSE 开根号，单位回到房价单位，更便于解释模型平均预测偏差的大致量级。

3. MAE：平均绝对误差

MAE = (1/n) × Σ|ŷᵢ - yᵢ|

MAE 表示预测值与真实值之间的平均绝对差距。它比 MSE 更不容易被极端误差影响，解释也更直观。

4. R²：决定系数

R² = 1 - SS_res / SS_tot

R² 表示模型对房价变化的解释能力。R² 越接近 1，说明当前特征对 MEDV 的线性解释能力越强。

  • R² 接近 1：拟合效果较好。
  • R² 接近 0：模型效果接近直接预测平均房价。
  • R² 小于 0：模型可能比平均值预测还差。

5. 指标仪表盘

在可视化界面中，RMSE、MAE、R² 可以通过仪表盘或曲线实时展示。随着训练进行，RMSE 和 MAE 通常应下降，R² 通常应上升。

指标方向适合回答的问题
MSE越小越好整体平方误差是否下降？
RMSE越小越好预测误差大约有多大？
MAE越小越好平均绝对偏差是多少？
R²越接近 1 越好当前特征能解释多少房价变化？
不同特征训练出的模型指标不同。比较 RM、LSTAT、PTRATIO 等特征的 R² 和 RMSE，可以判断哪个单特征更适合做简单线性回归。
