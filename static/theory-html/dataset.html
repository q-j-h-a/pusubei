数据探索与特征工程 (EDA & Preprocessing)

在建立任何机器学习模型之前，对数据进行探索性数据分析（Exploratory Data Analysis, EDA）和深度清洗是不可或缺的步骤。由于线性模型对数据的分布极为敏感，我们需要深入理解哪些数据对房价起决定性作用，并剔除其中的“杂质”。

1. 全局特征相关性分析 (Correlation Ranking)

面对波士顿房价数据集中的 13 个多维特征。我们不必深陷于晦涩的变量定义。借助统计学中的皮尔逊相关系数（Pearson Correlation）我们可以快速理清数据间的脉络，透视隐藏在数值背后的驱动逻辑。

皮尔逊系数可以量化各特征与目标房价（MEDV）之间的线性依赖程度。系数越接近 1 或 -1，说明该特征对预测房价的影响越显著：

图 1：各特征与目标房价(MEDV)的皮尔逊相关系数排名（蓝色代表推高房价，红色代表拉低房价）

通过上图我们可以极其直观地得出业务结论：房间数（RM）是促使房价升值的绝对主力；而底层弱势群体比例（LSTAT）、师生比（PTRATIO，即老师稀缺度）以及空气污染指数（NOX）则是导致房价暴跌的最强负面因素。

2. 数据清洗三步曲：从原始数据到纯净矩阵

找出了核心特征后，我们不能直接把原始数据生搬硬套进算法。真实世界的数据往往是肮脏的，我们必须执行以下三个核心清洗步骤：

步骤一：缺失值处理 (Missing Values)

数据集中可能会因为采集遗漏导致出现空值（NaN）。由于矩阵乘法无法对空值进行运算，我们必须对其进行处理。常见的策略包括：删除含有空值的行，或者用该列的均值/中位数进行填充（Imputation）。

步骤二：异常值裁剪 (Outliers Handling)

异常值就像是数据里的“钉子”，会强行把线性回归的拟合直线拉偏。例如，波士顿数据集中部分异常昂贵的豪宅被人工硬性截断封顶在了 50 万美元。如果不对这些极端的离群点进行过滤或盖帽处理（Clipping），模型的整体准确度将被严重拖累。

步骤三：特征标准化缩放 (Z-score Scaling)

不同特征的量纲（Scale）差异巨大。例如 NOX（污染浓度）通常在 0.4 左右，而 TAX（财产税率）高达 700。这会导致损失函数的误差曲面呈现极度拉伸的形态，使得后续的优化算法极其难以收敛。

我们必须将所有特征映射到均值为 0，标准差为 1 的同一尺度空间中：

$$ x_{ij}^{(norm)} = \frac{x_{ij} - \mu_j}{\sigma_j} $$
⚠️ 实验避坑指南：数据泄露 (Data Leakage)
在计算均值 $\mu_j$ 和标准差 $\sigma_j$ 时，必须仅使用训练集（Training Set）的数据。测试集的缩放也必须使用训练集的均值。如果违规使用了全量数据计算均值，会导致未来数据的信息提前“泄露”给模型，造成预测能力的虚高。

3. 核心代码实现：基于 Pandas 的数据清洗流水线

依托 Python 的 Pandas 和 NumPy 库，我们可以极其优雅地完成上述复杂的清洗工作：

import numpy as np
import pandas as pd

def clean_and_preprocess_data(df):
    """
    数据清洗与预处理流水线
    """
    # ---------------- 步骤一：缺失值处理 ----------------
    # 检查每列的缺失值数量
    # print(df.isnull().sum())
    
    # 策略：用该列的均值填充缺失值 (In-place 替换)
    df.fillna(df.mean(), inplace=True)
    
    # ---------------- 步骤二：异常值处理 ----------------
    # 策略：剔除目标房价(MEDV)刚好等于上限 50.0 的人工截断数据
    # 这能显著提升线性模型在正常区间的拟合度
    df = df[df['MEDV'] < 50.0]
    
    # 分离特征矩阵 X 和 标签向量 y
    X = df.drop('MEDV', axis=1).values
    y = df['MEDV'].values
    
    return X, y

def scale_features(X_train, X_test):
    """
    ---------------- 步骤三：特征标准化 ----------------
    注意：均值和标准差仅从训练集中提取，以防止数据泄露
    """
    mu = np.mean(X_train, axis=0)
    sigma = np.std(X_train, axis=0) + 1e-8 # 加上极小值防止除零报错
    
    X_train_norm = (X_train - mu) / sigma
    X_test_norm = (X_test - mu) / sigma
    
    return X_train_norm, X_test_norm
