思考拓展

本部分用于引导学生从简单线性回归出发，进一步思考模型适用条件、过拟合问题和更复杂的回归模型。

1. 一个特征真的足够预测房价吗？

简单线性回归只使用一个特征，例如 RM 或 LSTAT。它适合教学和解释，但真实房价往往受到多个因素共同影响。因此单特征模型可能无法捕捉全部房价变化。

简单线性回归适合理解“特征与标签的基本关系”，但不一定是最终预测效果最好的模型。

2. 当特征数量远大于样本数量时会怎样？

如果特征数量远大于样本数量，模型可能过度依赖训练数据中的偶然规律，导致在训练集上表现很好，但在新数据上表现很差。这就是过拟合风险。

  • 模型参数过多，容易记住训练样本。
  • 不同特征之间可能存在强相关，导致参数不稳定。
  • 训练误差低，不代表预测新样本也准确。

3. 如何判断模型是否过拟合？

可以把数据划分为训练集和测试集，对比两部分的指标。

现象可能含义
训练集误差低，测试集误差也低模型泛化较好
训练集误差低，测试集误差高可能过拟合
训练集误差高，测试集误差也高可能欠拟合

4. 过拟合有哪些解决方法？

  • 增加训练数据量。
  • 减少无关特征或弱相关特征。
  • 使用正则化方法，例如 Ridge 或 Lasso。
  • 使用交叉验证选择更合适的模型。
  • 控制模型复杂度，避免盲目增加特征或高阶项。

5. 可继续拓展的方向

拓展方向说明
多元线性回归把多个特征同时输入模型，比较与简单线性回归的效果差异
多项式回归用于拟合非线性关系，例如 LSTAT 与 MEDV 可能不是严格直线
Ridge 回归加入 L2 正则化，缓解参数过大和共线性问题
Lasso 回归加入 L1 正则化，可用于特征选择
交叉验证更稳定地评估模型泛化能力
可以把本实验看作回归模型学习的起点：先理解一个特征和一条直线，再逐步扩展到多个特征、非线性关系和正则化模型。
