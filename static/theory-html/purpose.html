实验目的

本实验通过 Boston Housing 房价预测案例，帮助学生理解简单线性回归的数学原理、几何意义、优化方法和评价指标。

1. 理解线性回归的数学原理

简单线性回归假设一个输入特征 x 与目标值 y 之间可以用一条直线近似表示：

ŷ = wx + b

其中，w 表示斜率，b 表示截距，ŷ 表示模型预测的房价中位数。模型训练的目标是找到合适的 w 和 b，使预测值尽可能接近真实 MEDV。

2. 理解线性回归的几何意义

在二维坐标系中，横轴是某个房价特征，例如 RM，纵轴是 MEDV。每个样本都是一个点，线性回归要寻找一条直线，让这条线尽可能贴近所有样本点。

样本点：(x, MEDV)
回归线：ŷ = wx + b
残差：真实值 y 与预测值 ŷ 的差距

3. 掌握最小二乘法和梯度下降

最小二乘法从整体误差最小的角度直接求解最优直线；梯度下降则从初始参数开始，按照损失函数下降最快的方向一步步更新参数。

方法核心思想教学重点
最小二乘法让误差平方和最小理解最优直线的数学含义
梯度下降反复更新 w 和 b，使 MSE 逐渐降低理解模型如何一步步学习

4. 学会评估回归模型性能

训练完成后，需要用评价指标判断模型是否有效。本实验重点关注 MSE、RMSE、MAE 和 R²。

  • MSE：均方误差，越小越好。
  • RMSE：均方根误差，单位更直观。
  • MAE：平均绝对误差，表示平均预测偏差。
  • R²：决定系数，表示模型对房价变化的解释能力。
学完本实验后，学生应能够解释一条回归线是怎样产生的、为什么损失会下降、学习率如何影响训练，以及如何判断一个单特征模型是否可靠。
