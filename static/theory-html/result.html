实验结果：白盒模型之美与完整闭环

1. 揭开黑盒：如何向业务方解释模型？

相比于深度神经网络那种无法解释的“黑盒”，线性回归被誉为最优雅的白盒模型（White-box Model）。这是因为我们求出的最优参数矩阵 $\mathbf{w}$ 具有极其明确的业务和经济学含义。

由于我们在特征工程阶段（dataset.html）已经对数据进行了标准化（Z-score），消除了量纲的影响，因此权重的绝对值大小直接代表了该特征对房价的影响力（Feature Importance），而正负号则代表了相关性的方向。

图 1：模型最终学到的特征权重（蓝色代表促使房价升值，红色代表导致房价贬值）

数据科学家的洞察报告：

  • 最核心的升值因素： RM (房间数) 的权重最高。这符合常识：在波士顿，房子越大、房间越多，价格必然越贵。
  • 最核心的贬值因素： LSTAT (弱势群体比例) 具有最大的负权重。这从社会学角度反映了社区贫困率对该区域房产价值的严重负面打击。
  • 环境与教育的折价： NOX (一氧化氮污染) 和 PTRATIO (师生比，数值越大说明老师越少) 同样拉低了房价，说明买房者愿意为好空气和优质学区支付溢价。
⚠️ 统计学陷阱：相关性 $\neq$ 因果性 (Correlation does NOT imply Causation)
需要极度警惕的是，线性回归的权重只能说明两个变量之间存在高度的“统计学相关”，而不能证明“因果关系”。例如，我们不能简单地得出“故意在小区里建几个大房子（人为提高 RM）就能带动整个小区房价上涨”的荒谬结论。因果推断需要借助更复杂的实验设计（如 A/B 测试）才能严谨证明。

2. 实验大结局：完整的训练闭环代码

现在，我们将前几个页面的零散代码片段，组装成一个可以端到端运行的完整训练函数。这段代码就是你本次实验的核心产出：

import numpy as np

def train_linear_regression(X_train, y_train, epochs=1000, learning_rate=0.01):
    """
    完整的线性回归模型训练闭环
    """
    m, n_features = X_train.shape
    
    # 1. 吸收偏置项：在 X 最左侧拼接一列全为 1 的常数
    X_train_bias = np.c_[np.ones((m, 1)), X_train]
    
    # 2. 参数初始化：随机生成形状为 (n_features + 1, 1) 的权重矩阵
    w = np.random.randn(n_features + 1, 1) * 0.01
    
    # 用于记录损失下降历史
    loss_history = []
    
    # 3. 梯度下降迭代
    for epoch in range(epochs):
        # 前向传播：计算预测值
        y_pred = np.dot(X_train_bias, w)
        
        # 计算当前的 MSE 损失
        loss = (1 / (2 * m)) * np.sum((y_pred - y_train) ** 2)
        loss_history.append(loss)
        
        # 反向传播：计算梯度并更新参数
        gradient = (1 / m) * np.dot(X_train_bias.T, (y_pred - y_train))
        w = w - learning_rate * gradient
        
        # 打印训练进度
        if epoch % 100 == 0:
            print(f"Epoch {epoch} | MSE Loss: {loss:.4f}")
            
    return w, loss_history
