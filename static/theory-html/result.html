预期成果

完成本实验后，学生应能够手动实现简单线性回归算法，理解梯度下降原理，并完成一个可交互的房价预测 Demo。

1. 算法实现成果

  • 能够手动实现 ŷ = wx + b 的预测过程。
  • 能够手动计算 MSE 损失函数。
  • 能够根据梯度下降公式更新 w 和 b。
  • 能够记录每一轮训练的 w、b、Loss 和评价指标。
  • 能够不调用高级模型库，完成从零训练简单线性回归。

2. 可视化成果

可视化模块预期作用
特征与 MEDV 散点图观察单特征线性关系
回归线动态更新观察模型训练过程
Loss 曲线观察 MSE 是否下降
Loss 等高线图观察参数在损失地形中的移动路径
w、b 参数轨迹观察参数如何逐渐收敛
指标仪表盘实时查看 RMSE、MAE、R²

3. 交互 Demo 成果

学生应能够完成一个房价预测交互 Demo：

  • 选择一个输入特征，例如 RM。
  • 选择原始特征或标准化特征。
  • 输入新的特征值。
  • 系统自动代入训练好的模型进行预测。
  • 显示预测 MEDV、计算过程和预测点可视化。

4. 学习理解成果

学生应能够用自己的语言解释：

简单线性回归通过一条直线描述单个特征与房价 MEDV 的关系。训练过程通过最小化 MSE 调整 w 和 b。学习率会影响训练速度和稳定性，RMSE、MAE 和 R² 可以从不同角度评价模型效果。
最终成果应包含代码、图表、训练过程、评价指标和预测交互，而不只是静态的预测结果。
