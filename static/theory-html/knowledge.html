前置知识储备：数学、机器学习与工具栈

1. 数学原理：寻优与数据标准化

导数与偏导数 (Derivatives & Partial Derivatives)

你可以把导数理解为悬崖上某一点的“倾斜程度”（斜率），它告诉我们函数值变大还是变小的趋势。

什么是偏导数？想象你被蒙上双眼站在一个凹凸不平的山坡上，你要往下走找谷底。偏导数就是指：如果你只允许沿着正前方的 X 方向走一步，或者只允许沿着右侧的 Y 方向走一步，各自的坡度分别是多少。在我们的实验中，模型需要不断调整参数来减少“预测误差”。导数会精确地给模型指明“朝哪个方向调整，误差减小得最快”。

均值与标准差 (Mean & Standard Deviation)

均值代表数据的中心位置，而标准差代表数据的离散程度（波动大小）。

在波士顿房价数据集中，不同特征的单位差异极大。比如“房屋面积”动辄几百平方米，而“房间数”只是个位数。如果不做处理，模型在学习时会被数值庞大的特征带偏。我们将利用均值和标准差对数据进行“标准化”处理，让所有特征都站在同一起跑线上。

2. 机器学习核心概念

监督学习 (Supervised Learning)

这就像是学生在做带有参考答案的模拟题。我们给计算机喂入大量的“历史房屋数据（题目）”以及它们对应的“真实房价（答案）”，让它从中寻找规律。等它学会后，面对一套未知价格的新房子，它就能给出合理的预测。这就是监督学习的本质。

特征 (Feature) 和标签 (Label)

  • 特征（通常用矩阵 $\mathbf{X}$ 表示）：就是影响房价的各种因素，比如房子的面积、有几个房间、离市中心有多远。这些是我们输入给模型的“线索”。
  • 标签（通常用向量 $\mathbf{y}$ 表示）：就是我们要预测的目标。在这个实验里，标签就是真实的房价。
核心区分：回归 (Regression) vs 分类 (Classification)

这两个任务的关键区别，不在于“预测”本身，而在于输出结果的类型：

  • 分类：输出是离散类别。比如判断一张图片是猫还是狗，结果只能落在有限几个选项里。
  • 回归：输出是连续数值。本实验中的房价就是连续变化的数值，例如 23.5 万、24.1 万。线性回归正是解决这类问题的经典算法。

3. Python 编程基础与可视化库

在代码实战部分，我们将高度依赖两个强大的 Python 库：NumPy 和 Matplotlib。

NumPy 计算库：向量化运算

实验中我们需要同时处理数百套房屋的数据。如果采用普通的 Python for 循环逐条计算，效率会非常低下。更高效的做法是将数据组织为矩阵和向量（即多维数组 ndarray），借助 NumPy 的向量化运算一次性完成批量数学操作。

Matplotlib 绘图库：可视化数据与模型

我们将使用 Matplotlib 来直观地展示数据分布和模型的拟合效果：

  • 散点图 (Scatter Plot)：在二维坐标平面上绘制样本点。例如以房间数量为横轴、房价为纵轴，每个点代表一套真实房屋的数据。
  • 直线图 (Line Plot)：实验的最终目标是让程序拟合出一条最优直线，使该直线尽可能穿过散点图所呈现的数据群，从而刻画变量间的近似关系。
