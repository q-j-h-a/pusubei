前置知识

本实验需要具备少量 Python 编程基础和基础数学知识，重点理解导数、偏导数、损失函数和二维坐标系中的直线关系。

1. Python 基础编程

学生需要能够阅读和编写基本 Python 代码，包括变量、函数、循环、列表和数组操作。

  • 能够使用变量保存特征值、标签值、参数 w 和 b。
  • 能够使用循环执行多轮训练。
  • 能够使用函数封装预测、损失计算和参数更新过程。
  • 能够使用 NumPy 完成数组计算。

2. 一次函数与直线

简单线性回归的模型本质上就是一条直线：

y = wx + b
  • w：斜率，决定直线倾斜方向和倾斜程度。
  • b：截距，决定直线整体上下位置。
  • x：输入特征，例如 RM。
  • y：标签值，例如 MEDV。

3. 导数和偏导数

导数可以理解为函数在某一点变化的快慢。在线性回归训练中，我们关心的是损失函数对参数 w 和 b 的变化方向。

如果 ∂Loss/∂w > 0，说明 w 增大可能使 Loss 增大，需要减小 w
如果 ∂Loss/∂w < 0，说明 w 增大可能使 Loss 减小，可以增大 w

偏导数表示在多个变量中，只研究其中一个变量变化时函数如何变化。梯度下降更新 w 和 b 时，就会分别用到对 w 和 b 的偏导。

4. 均值、标准差与标准化

实验界面中支持原始特征和标准化特征切换。标准化用于把不同特征调整到相似尺度，方便观察和训练。

x_standardized = (x - mean) / std

标准化不会改变特征与 MEDV 的整体相关方向，但会改变横轴数值范围，也会改变参数 w 和 b 的数值大小。

对初学者来说，先把线性回归理解成“在散点图中找一条合适的直线”，再理解梯度下降和损失函数，会更容易。
